# HiddenProtectedWorkProfile
Hidden Profile owner Android app.

HiddenProtectedWorkProfile is copy of ProtectedWorkProfile app, which, unlike original, is hidden in work profile and launched via notification.
Since app hides all other apps when screen turns off, hiding itself leads to hiding entire profile from launcher.

This app creates work profile that will be frozen and that hide apps when your screen turns off. This app not visible in profile.
